## SmallOS 系統調用表 (System Call Table)

| 編號 (EAX) | 功能名稱 | 參數 1 (EBX) | 參數 2 (ECX) | 返回值 (EAX) | 說明 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **0** | `SYS_EXIT` | 退出狀態碼 | - | - | 終止當前用戶進程 (未實現) |
| **1** | `SYS_PRINT` | 字符串地址 | - | 打印字符數 | 在內核終端輸出字符串 |
| **2** | `SYS_TIME` | - | - | 系統滴答數 | 獲取自啟動以來的 Timer Ticks(未實現) |
| **3** | `SYS_SLEEP` | 毫秒數 | - | 0 | 讓當前進程睡眠一段時間(未實現) |
| **4** | `SYS_MALLOC` | 申請字節數 | - | 內存首地址 | 用戶態堆內存分配(未實現) |
